# Thực hành form đăng nhập

## HỌ VÀ TÊN: **NHỮ TRUNG HẢI**

## MSV: **23810310247**

## Tên bài thực hành

**Thực hành form đăng nhập**

## Mô tả

Project PHP kết nối MySQL, gồm chức năng đăng ký tài khoản, đăng nhập, đăng xuất và trang chào mừng sau khi đăng nhập thành công.

## Cơ sở dữ liệu

- Database: `thuc_hanh_dang_nhap_db`
- Bảng: `users`

## Cách chạy

1. Bật Apache và MySQL trong XAMPP.
2. Truy cập `http://127.0.0.1:5000/thuc_hanh_dang_nhap/`
